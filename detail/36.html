    mm: Fix mremap not considering huge pmd devmap
    
    The original code in mm/mremap.c checks huge pmd by:
    
                    if (is_swap_pmd(*old_pmd) || pmd_trans_huge(*old_pmd)) {

    changes the condition to include the case.
